Welcome to the API that can be used for online courses.

Here's a list of all routes:

A. FOR USERS:

1. POST on '/api/users/' to register user

2. GET on '/api/users/' for admin to see user list

3. POST on '/api/users/login' to login

4. GET on '/api/users/profile' to get logged user profile

5. PUT on '/api/users/profile' to update logged user profile

6. DELETE on '/api/users/:id' for admins to delete a user

7. GET on '/api/users/:id' for admins to a user profile

8. PUT on '/api/users/:id' for admins to make a user an admin

9. PUT on '/api/users/:id/educator' to make a user an educator

B. FOR ORDERS:

1. POST on '/api/orders/' to addOrder

2. GET on '/api/orders' fro admins to get all orders

3. GET on '/api/orders/myorders' to get user orders passing the user id

4. GET on '/api/orders/:id' to get an order with a certain id

5. PUT on 'api/orders/:id/pay' to update the order to paid

C. FOR COURSES:

1. GET on '/api/courses/' to get list of all courses

2. POST on '/api/courses/' for educator to post a course

3. POST on '/api/courses/:id/review' to post a review

4. GET on '/api/courses/top' to get top courses

5. GET on '/api/courses/:id' to get a specific course

6. DELETE on '/api/courses/:id' for educator to delete a course

7. PUT on '/api/courses/:id' for educator to update a course

8. DELETE on '/api/courses/:id/:dir' for educator delete a subDirectory from a course

9. POST on '/api/courses/:id/:dir' to add a subDirectory to a course

10. PUT on '/api/courses/:id/:dir' to update a subDirectory in a cpurse

11. POST on '/api/courses/:userId/cart' to add a course to a user's cart

12. POST on '/api/courses/:userId/whishlist' to add a course to a user's whishlist

13. POST on '/api/courses/:userId/learning' to add a course to a user's learning list

14. POST on '/api/courses/:userId/courses' to add a course to a user's courses if he's an educator

D. FOR VIDEO UPLOAD:

1. POST on '/api/uploads' containing the videos to upload
